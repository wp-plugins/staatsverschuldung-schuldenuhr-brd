== Description ==

Easy to customize `Bundesrepublik Deutschland Schuldenuhr` mit pro-Kopf Verschuldung as Wordpress-Widget

This wordpress plugin is maintained by [https://www.coininvest.com/](https://www.coininvest.com/).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `ism-schuldenuhr` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Appearance > Widgets
4. To activate the widget drag it to a sidebar or click on it.

== Frequently Asked Questions ==

= How to change the layout =

Change the default basic style via CSS which can be found in the CSS subfolder of the plugin.
<plugin-folder>/ism-schuldenuhr/css/default.css

= How to change the initial numbers =

Edit the configuration values here:
<plugin-folder>/ism-schuldenuhr/js/ism-schuldenuhr.json
